trickle

Runtime type observability for JavaScript and Python. Run your code, get types in your IDE automatically.

# In Jupyter or any Python script:
%load_ext trickle

model = nn.Linear(784, 10)  # → Linear(7850 params 30.7 KB)
x = torch.randn(32, 784)    # → Tensor[32, 784] float32
loss.backward()             # → model: |∇|=5.32

Get Started in 30 Seconds

Python / Jupyter

pip install trickle-observe
Then: %load_ext trickle or import trickle.auto

JavaScript / TypeScript

npm i trickle-observe
Then: trickle run node app.js
