# User Agent Notepad

Write your project progress, trickle observations, and gaps found here. This persists across sessions.

## Format

Use dated entries. Record what project you're building, what trickle showed you, and where it fell short.

```
## YYYY-MM-DD

### Project
- What I'm building: ...
- Codebase: ...

### Trickle usage
- Commands run and what they showed
- What was helpful
- What was wrong/missing (file issues.json for these)

### Project progress
- What I accomplished
- What's next
```
