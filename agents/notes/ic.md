# IC Notepad

Write your progress, debugging notes, and learnings here. This persists across sessions.

## Format

Use dated entries. Record what you're working on, what you tried, what worked/failed.

```
## YYYY-MM-DD

### Working on
- Focus point: ...
- Issue: ...

### Progress
- ...

### Blocked / needs help
- ...

### Learnings
- Things discovered about the codebase that future ICs should know
```
